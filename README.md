# AI-by-hand

AI by Hand in code

Notes of the AI by Hand Workfbook into Python and Latex


## Setup

1. Create and activate the virtual environment:
   ```zsh
   pyenv virtualenv aibyhand
   pyenv activate aibyhand
   ```

2. Install dependencies from `pyproject.toml`:
   ```zsh
   uv pip install -r pyproject.toml
   ```
